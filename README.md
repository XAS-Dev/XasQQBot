# XAS-QQBot-v2

XAS Minecraft 服务器 QQ 群机器人

## 功能

- [x] AnimeTrace 动漫人物识别
- [x] 萌娘百科搜索
- [x] Github 项目卡片
- [ ] Github commits 消息推送
- [x] ChatGPT 聊天
- [x] 获取 MC 服务器状态

## 如何使用

### 直接运行

使用 `pip install .` 或 `pdm install` 安装依赖

使用 `nb run --reload` 运行机器人

### 使用Docker

建议使用 docker compose
